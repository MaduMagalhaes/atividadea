1. Introdução
    1. O que são arrays?
      1. Variável de JavaScript
      2. Pode conter mais de um valor por vez

    2. Função de uma array
      1. Armazenamento de valores
      2. Acessar vários valores em um único índice

    3. Conceito de array
      1. É um objeto com um construtor único, com uma sintaxe literal
      2. Com um conjunto adicional de propriedades e de métodos herdados de um protótipo de Array

  2. Utilizações
    1. Lista
      1. Pode-se usar uma array para substituir a utilização de várias variáveis
      2. Armazena-se vários valores em apenas uma array, obtendo assim uma lista

    2. Organização
      1. Possibilidade de mânter os itens mais organizados
      2. Auxilia na organização do código

    3. Array em array
      1. Uma array pode ter outra array dentro dela

  3. Métodos
    1. Adição
      1. Para adicionar um item na array utilizamos 'nomeDaArray'+'.push'
      2. Exemplo: Para adicionarmos 'um' na array numeros fazemos um.push(numeros)

    2. Remoção do último item
      1. Para removermos o último item de uma array utilizamos 'nomeDaArray'+'.pop'
      2. Exemplo: Para removermos 'um' na array numeros fazemos numeros.pop

    3. Base
      1. Sua estrutura base consiste na utilização de cochetes []
      2. Sem isso não é considerada uma array

  4. Estrutura
    1. Como declarar uma array?
      1. A maneira mais comum é criando uma variável
      2. var numeros = [um, dois, tres]

    2. Elementos
      1. Nome da array
      2. Itens

    3. Leitura
      1. O primeiro item da array sempre será o zero
      2. Ela segue em ordem crescente
